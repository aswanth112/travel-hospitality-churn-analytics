#  Travel Hospitality Chain Analysis

##  Project Overview

This project focuses on analyzing hospitality booking data to understand customer behavior, booking patterns, and business trends. The goal is to prepare clean and structured data for further analysis and visualization.

##  Objective

- Clean and preprocess raw hospitality booking data.
- Handle missing values and duplicate records.
- Standardize data formats for analysis.
- Prepare the dataset for Exploratory Data Analysis (EDA) and dashboard creation.

##  Tools Used

- Python
- Pandas
- Jupyter Notebook

##  Project Workflow

### Data Cleaning
- Imported dataset using Pandas.
- Checked dataset structure and data types.
- Identified and handled missing values.
- Removed duplicate records.
- Cleaned and standardized data where required.
- Prepared the dataset for further analysis.

##  Project Files

- `Travel_Hospitality_Analysis.ipynb` – Jupyter Notebook containing data cleaning process.
- `README.md` – Project documentation.
